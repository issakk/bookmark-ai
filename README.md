# AI Bookmark Manager

智能书签管理浏览器插件 - 使用 AI 自动分类、整理和搜索您的书签

## 功能特性

### 核心功能
- 📚 **书签同步**：自动同步浏览器所有书签
- 🤖 **AI 智能分类**：使用 OpenAI 自动为书签分类和打标签
- 🌲 **树形结构**：在侧边栏以树形结构查看所有书签
- 🔍 **智能筛选**：通过 AI 归类后的结构快速筛选书签
- ⚙️ **书签管理**：完整的书签管理页面，支持调整、删除、添加
- ⚡ **快速保存**：一键保存当前页面为书签

### 增强功能
- 🔎 **智能搜索**：支持模糊搜索、标签搜索、AI 语义搜索
- 📊 **数据统计**：书签数量、分类分布、访问统计
- 🏷️ **自定义标签**：手动添加和管理标签
- 🎨 **主题切换**：深色/浅色模式
- ⌨️ **快捷键**：快速访问常用功能
- 📤 **导入导出**：支持 JSON/HTML 格式
- 🧹 **智能清理**：检测失效链接和重复书签

## 安装方法

### Chrome/Edge
1. 下载或克隆此项目
2. 打开浏览器扩展管理页面 (`chrome://extensions/`)
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `bookmark-ai-extension` 文件夹

## 配置

1. 点击扩展图标或打开选项页面
2. 输入您的 OpenAI API Key
3. 选择使用的模型（推荐 gpt-4o-mini 或 gpt-3.5-turbo）
4. 点击保存

## 使用方法

### 快捷键
- `Ctrl+Shift+S` (Mac: `Cmd+Shift+S`) - 快速保存当前页面
- `Ctrl+Shift+B` (Mac: `Cmd+Shift+B`) - 打开书签侧边栏

### 侧边栏
- 点击扩展图标打开侧边栏
- 浏览树形结构的书签
- 使用搜索框快速查找
- 点击书签直接打开

### 管理页面
- 右键点击扩展图标 → 选项
- 或访问 `chrome://extensions/` 找到本插件点击"详情" → "扩展程序选项"

## 开发

```bash
# 安装依赖
npm install

# 开发模式（监听文件变化）
npm run dev

# 构建生产版本
npm run build
```

## 隐私说明

- 所有书签数据存储在本地浏览器中
- OpenAI API Key 加密存储在本地
- 仅在用户主动触发 AI 功能时才会调用 OpenAI API
- 不会收集或上传任何用户数据到第三方服务器
