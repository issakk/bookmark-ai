# 安装和使用检查清单

## 📋 安装前检查

### 环境要求
- [ ] Chrome/Edge 浏览器版本 120+
- [ ] 有 OpenAI API Key（或准备注册）
- [ ] 网络连接正常

### 文件完整性
- [ ] `manifest.json` 存在
- [ ] `background.js` 存在
- [ ] `utils/` 文件夹包含 3 个 JS 文件
- [ ] `scripts/` 文件夹包含 3 个 JS 文件
- [ ] `styles/` 文件夹包含 3 个 CSS 文件
- [ ] HTML 文件存在（sidebar.html, options.html, manage.html）

## 🎨 图标生成

### 方法 1：使用 HTML 工具（推荐）
- [ ] 在浏览器中打开 `generate-icons.html`
- [ ] 点击"下载全部"按钮
- [ ] 将 4 个 PNG 文件移动到 `icons/` 文件夹
- [ ] 确认文件名：icon16.png, icon32.png, icon48.png, icon128.png

### 方法 2：使用脚本（需要 ImageMagick）
- [ ] 安装 ImageMagick: `brew install imagemagick`
- [ ] 运行: `chmod +x generate-icons.sh && ./generate-icons.sh`
- [ ] 确认 `icons/` 文件夹包含 4 个 PNG 文件

## 🔧 安装扩展

### Chrome/Edge
- [ ] 打开 `chrome://extensions/` 或 `edge://extensions/`
- [ ] 开启右上角"开发者模式"
- [ ] 点击"加载已解压的扩展程序"
- [ ] 选择 `bookmark-ai-extension` 文件夹
- [ ] 扩展图标出现在工具栏
- [ ] 没有错误提示

## ⚙️ 配置 OpenAI API

### 获取 API Key
- [ ] 访问 https://platform.openai.com/api-keys
- [ ] 注册/登录 OpenAI 账号
- [ ] 创建新的 API Key
- [ ] 复制 API Key（格式：sk-...）

### 配置扩展
- [ ] 点击扩展图标（自动打开配置页面）
- [ ] 粘贴 API Key
- [ ] 选择模型：`gpt-4o-mini`（推荐）
- [ ] 点击"测试连接"
- [ ] 显示"✅ 连接成功"
- [ ] 点击"保存设置"
