# 更新日志

## v1.0.1 (2024-11-28)

### ✨ 新功能

1. **基于用户书签目录的分类**
   - 书签现在优先按照用户的浏览器文件夹结构分类
   - 如果用户已经有自己的文件夹组织，会优先显示文件夹名称
   - AI 分类作为补充，用于没有文件夹的书签

2. **删除失效书签功能**
   - 新增"查找失效书签"功能
   - 自动检测以下类型的失效书签：
     - 本地链接（localhost, 127.0.0.1）
     - 局域网链接（192.168.x.x, 10.x.x.x）
     - 文件链接（file://）
     - Chrome 内部链接（chrome://）
     - 临时/测试链接（temp, test, example.com）
   - 支持一键批量删除失效书签

3. **完整的用户使用说明文档**
   - 新增 `user-guide.html` 用户使用说明
   - 包含详细的安装、配置、使用教程
   - 包含常见问题解答
   - 精美的排版和样式

### 🎨 界面优化

1. **书签管理页面美化**
   - 增强书签卡片样式，添加渐变背景
   - 添加左侧装饰条动画效果
   - 优化悬停和选中状态的视觉反馈
   - 增加更多颜色变量和设计 token
   - 提升整体视觉层次感

2. **更好的视觉效果**
   - 卡片阴影效果增强
   - 过渡动画更流畅
   - 边框和圆角更精致

### 🐛 问题修复

1. **主题设置**
   - 修复主题默认值，现在默认为"跟随系统"
   - 用户首次安装时会自动跟随系统主题

2. **配置页面链接**
   - 更新 GitHub 链接为：https://github.com/simonguo/bookmark-ai
   - 更新反馈邮箱为：simonguo.2009@gmail.com
   - 添加用户使用说明文档链接

### 📝 文档更新

- 新增完整的用户使用说明（user-guide.html）
- 更新 README.md 中的项目信息
- 添加更新日志（CHANGELOG.md）

### 🔧 技术改进

1. **书签数据结构增强**
   - 书签对象新增 `folderPath` 字段（完整路径）
   - 书签对象新增 `folderName` 字段（直接父文件夹名）
   - 更好地支持基于文件夹的分类

2. **批量删除功能**
   - 新增 `deleteBookmarks()` 方法支持批量删除
   - 返回详细的成功/失败结果
   - 更好的错误处理
