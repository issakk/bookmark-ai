## 🚀 使用指南

### 快速开始（5 分钟）

**1. 生成图标**
```bash
# 方法 1：浏览器
打开 generate-icons.html → 下载全部 → 移动到 icons/

# 方法 2：命令行（需要 ImageMagick）
chmod +x generate-icons.sh
./generate-icons.sh
```

**2. 安装扩展**
```
1. 打开 chrome://extensions/
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 bookmark-ai-extension 文件夹
```

**3. 配置 API**
```
1. 获取 OpenAI API Key
2. 点击扩展图标
3. 输入 API Key
4. 选择 gpt-4o-mini
5. 测试连接
6. 保存设置
```

**4. 开始使用**
```
1. 保存书签：Cmd+Shift+S
2. AI 分类：点击"AI 分类"
3. 搜索书签：输入关键词
4. 管理书签：打开管理页面
```

## 🐛 已知限制

### 技术限制
1. **浏览器兼容性**
   - 仅支持 Chromium 内核浏览器
   - 不支持 Firefox（需要适配）

2. **性能限制**
   - 大量书签（5000+）可能需要优化
   - 建议使用虚拟滚动

3. **API 限制**
   - 需要网络连接
   - 有速率限制
   - 需要付费（虽然很便宜）

### 功能限制
1. 暂不支持云同步
2. 暂不支持团队协作
3. 暂不支持移动端
4. 暂不支持自定义 AI 模型
