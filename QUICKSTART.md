# 快速开始指南

## 5 分钟上手 AI Bookmark Manager

### 第一步：生成图标 (1 分钟)

1. 用浏览器打开 `generate-icons.html`
2. 点击"下载全部"按钮
3. 将 4 个 PNG 文件移动到 `icons/` 文件夹

### 第二步：安装扩展 (1 分钟)

1. 打开 Chrome/Edge，访问 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `bookmark-ai-extension` 文件夹
5. 完成！扩展图标出现在工具栏

### 第三步：配置 API (2 分钟)

1. 点击扩展图标，会自动打开配置页面
2. 输入你的 OpenAI API Key
   - 没有？去 [OpenAI Platform](https://platform.openai.com/api-keys) 创建一个
3. 选择模型：推荐 `gpt-4o-mini`（便宜又好用）
4. 点击"测试连接"确保配置正确
5. 点击"保存设置"

### 第四步：开始使用 (1 分钟)

1. **保存书签**
   - 浏览任意网页
   - 按 `Cmd+Shift+S` (Mac) 或 `Ctrl+Shift+S` (Windows)
   - 或点击侧边栏的"保存当前页面"按钮

2. **AI 分类**
   - 打开侧边栏（点击扩展图标）
   - 点击"AI 分类"按钮
   - 等待 AI 自动分类所有书签

3. **搜索书签**
   - 在侧边栏搜索框输入关键词
   - 勾选"AI 搜索"使用语义搜索
   - 立即找到相关书签

4. **管理书签**
   - 点击"管理页面"按钮
   - 批量编辑、删除、整理书签
   - 查找重复书签

## 常见问题

### Q: 没有 OpenAI API Key 怎么办？

A: 访问 [OpenAI Platform](https://platform.openai.com/api-keys) 注册并创建 API Key。新用户通常有免费额度。

### Q: API 调用会很贵吗？

A: 使用 `gpt-4o-mini` 模型，分类 100 个书签大约花费 $0.01-0.02，非常便宜。

### Q: 数据安全吗？

A: 完全安全！所有数据存储在本地浏览器，API Key 加密保存，不会上传到任何服务器。

### Q: 可以不用 AI 功能吗？

A: 可以！不配置 API Key 也能使用基础的书签管理、搜索、筛选功能。

### Q: 支持哪些浏览器？

A: 支持所有基于 Chromium 的浏览器：Chrome、Edge、Brave、Opera 等。

## 进阶使用

### 批量操作

1. 打开管理页面
2. 勾选多个书签
3. 点击"AI 分类选中项"或"删除选中项"

### 导出备份

1. 打开配置页面
2. 点击"导出数据"保存 JSON 文件
3. 或点击"导出书签 (HTML)"导出标准格式

### 查找重复

1. 打开管理页面
2. 点击"查找重复"
3. 查看并删除重复的书签
