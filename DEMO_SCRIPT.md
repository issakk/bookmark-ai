# 演示脚本

## 功能演示流程（5-10 分钟）

### 场景：展示如何使用 AI 书签管理器整理杂乱的书签

---

## 第一部分：安装和配置 (2 分钟)

### 1. 安装扩展
**演示步骤：**
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展图标出现在工具栏

**旁白：**
> "首先，我们来安装这个扩展。打开 Chrome 的扩展管理页面，开启开发者模式，加载我们的项目文件夹。安装完成后，你会看到一个蓝色的书签图标出现在工具栏上。"

### 2. 配置 OpenAI API
**演示步骤：**
1. 点击扩展图标（自动打开配置页面）
2. 输入 OpenAI API Key
3. 选择模型：`gpt-4o-mini`
4. 点击"测试连接"
5. 显示"✅ 连接成功"
6. 点击"保存设置"

**旁白：**
> "第一次使用时，我们需要配置 OpenAI API。输入你的 API Key，选择 GPT-4o Mini 模型，这是性价比最高的选择。点击测试连接，确保一切正常，然后保存设置。"

---

## 第二部分：基础功能演示 (3 分钟)

### 3. 快速保存书签
**演示步骤：**
1. 打开任意网页（如：https://react.dev）
2. 按快捷键 `Cmd+Shift+S`
3. 显示通知："保存成功"
4. 再打开另一个网页（如：https://vuejs.org）
5. 点击侧边栏的"保存当前页面"按钮
6. 再次显示保存成功

**旁白：**
> "保存书签非常简单。你可以使用快捷键 Cmd+Shift+S，或者点击侧边栏的保存按钮。看，书签立即被保存了。"

### 4. 查看书签列表
**演示步骤：**
1. 点击扩展图标打开侧边栏
2. 展示书签列表（显示多个未分类的书签）
3. 滚动浏览书签
4. 点击一个书签打开网页

**旁白：**
> "打开侧边栏，我们可以看到所有的书签。现在它们还没有分类，都在'未分类'文件夹下。让我们用 AI 来整理它们。"

---

## 第三部分：AI 智能分类 (2 分钟)

### 5. AI 批量分类
**演示步骤：**
1. 在侧边栏点击"AI 分类"按钮
2. 确认对话框："确定要对 X 个书签进行 AI 分类吗？"
3. 点击确定
4. 显示进度条："正在分类... (3/20) 15%"
5. 等待完成（可以加速播放）
6. 显示通知："分类完成"

**旁白：**
> "点击 AI 分类按钮，AI 会自动分析每个书签的内容，给它们分类和打标签。你可以看到进度条实时更新。这个过程可能需要一点时间，但完全自动化，不需要任何手动操作。"

### 6. 查看分类结果
**演示步骤：**
1. 刷新侧边栏
2. 展示书签已按分类分组：
   - 📁 技术 (8)
   - 📁 新闻 (3)
   - 📁 娱乐 (2)
   - 📁 购物 (4)
   - 📁 教育 (3)
3. 点击展开"技术"分类
4. 显示书签带有标签：
   - 🔖 React 官方文档 [前端] [React] [文档]
   - 🔖 Vue.js 官网 [前端] [Vue] [框架]

**旁白：**
> "分类完成！看，所有书签都被整齐地归类了。技术类、新闻类、娱乐类等等。每个书签还有详细的标签，比如这个 React 文档被标记为'前端'、'React'、'文档'。"

---

## 第四部分：搜索和筛选 (2 分钟)

### 7. 普通搜索
**演示步骤：**
1. 在搜索框输入"react"
2. 立即显示匹配结果
3. 清除搜索

**旁白：**
> "搜索功能非常强大。输入关键词，立即找到相关书签。"

### 8. AI 语义搜索
**演示步骤：**
1. 勾选"AI 搜索"
2. 输入"学习前端开发"
3. 显示加载中
4. 返回所有前端相关的书签（React、Vue、MDN 等）
5. 再试试"买东西"
6. 返回所有购物网站

**旁白：**
> "更酷的是 AI 语义搜索。我搜索'学习前端开发'，AI 理解我的意图，返回所有前端相关的资源。再试试'买东西'，它会找到所有购物网站。这比传统的关键词搜索智能多了。"

### 9. 分类筛选
**演示步骤：**
1. 在分类下拉框选择"技术"
2. 只显示技术类书签
3. 在标签下拉框选择"React"
4. 只显示带 React 标签的书签
5. 点击"清除筛选"恢复

**旁白：**
> "你还可以按分类和标签筛选。选择'技术'类别，只看技术相关的。再选择'React'标签，精确定位到 React 相关的书签。"

---

## 第五部分：管理功能 (2 分钟)

### 10. 打开管理页面
**演示步骤：**
1. 点击"管理页面"按钮
2. 打开新标签页，显示管理界面
3. 展示左侧工具栏和右侧书签列表

**旁白：**
> "如果需要更强大的管理功能，可以打开管理页面。这里有完整的书签管理工具。"

### 11. 批量操作
**演示步骤：**
1. 勾选几个书签
2. 显示"已选 3"
3. 点击"AI 分类选中项"
4. 等待完成

**旁白：**
> "你可以批量选择书签，然后进行批量操作，比如批量分类、批量删除等。"

### 12. 查找重复
**演示步骤：**
1. 点击"查找重复"按钮
2. 显示对话框："发现 2 组重复书签"
3. 展示重复的 URL
4. 可以选择删除重复项

**旁白：**
> "管理页面还能查找重复的书签。点击查找重复，它会列出所有重复的 URL，你可以选择删除多余的。"

### 13. 编辑书签
**演示步骤：**
1. 点击书签的编辑按钮
2. 打开编辑对话框
3. 修改标题、URL、分类、标签
4. 点击保存
5. 书签更新

**旁白：**
> "编辑书签也很简单。点击编辑按钮，修改任何信息，包括手动调整分类和标签。"

---

2. 滚动到"数据管理"区域
3. 点击"导出数据"
4. 下载 JSON 文件
5. 点击"导出书签 (HTML)"
6. 下载 HTML 文件

**旁白：**

### 15. 查看统计
**演示步骤：**
1. 在配置页面查看统计信息
2. 显示：
   - 书签总数：20
   - 已分类：20
   - 分类数量：5
   - 标签数量：15

**旁白：**
> "在配置页面，你可以看到详细的统计信息。书签总数、已分类数量、分类和标签的数量，一目了然。"

## 演示技巧

### 准备工作
1. 提前准备 20-30 个不同类型的书签
2. 确保 API Key 已配置且有额度
3. 清除浏览器缓存，确保干净的演示环境
4. 准备好录屏软件

### 演示注意事项
1. 操作要慢，给观众时间理解
2. 每个功能演示后暂停 1-2 秒
3. 使用鼠标高亮或圈出重点
4. AI 分类过程可以加速播放
5. 保持画面整洁，关闭无关标签页

### 可选的高级演示
1. 展示快捷键操作
2. 展示右键菜单
3. 展示主题切换
4. 展示移动端响应式（如果支持）
5. 展示错误处理（如 API Key 错误）
