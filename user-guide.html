User Guide

AI Bookmark Manager is an intelligent bookmark management extension that uses AI to automatically classify, organize, and search your bookmarks.

üöÄ Getting Started

Installation

  1. Open Chrome/Edge browser
  2. Go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked"
  5. Select the extension folder

Setup OpenAI API

  1. Visit OpenAI Platform
  2. Create an API Key
  3. Open extension settings
  4. Paste your API Key
  5. Select model (recommended: gpt-4o-mini)
  6. Click "Test Connection"
  7. Click "Save Settings"

üìñ Basic Usage

Save Bookmarks

  • Keyboard: Cmd+Shift+S (Mac) or Ctrl+Shift+S (Windows)
  • Sidebar: Click "Save Current Page" button
  • Browser: Use native bookmark function

View Bookmarks

  1. Click extension icon to open sidebar
  2. Bookmarks are organized by folders/categories
  3. Click folder to expand/collapse
  4. Click bookmark to open

Search Bookmarks

  • Keyword Search: Type in search box
  • AI Search: Enable "AI Search" for semantic search
  • Filter: Use category/tag dropdowns

ü§ñ AI Features

AI Classification

AI analyzes each bookmark and generates:

  • Category: Tech, News, Shopping, etc.
  • Tags: 3-5 relevant tags
  • Description: Brief summary

How to use:

  1. Open sidebar
  2. Click "AI Classify" button
  3. Wait for processing
  4. Bookmarks are automatically categorized
üí° Cost: Using gpt-4o-mini, classifying 100 bookmarks costs ~$0.01-0.02.

AI Semantic Search

Management Page

Click "Manage" button in sidebar to access:

  • Batch Operations: Select multiple bookmarks
  • Add/Edit: Manually manage bookmarks
  • Find Duplicates: Detect duplicate bookmarks
  • Find Invalid: Detect broken links
  • View Modes: List or grid view
  • Sort: By title, date, or URL

Data Management

In settings page:

  • Export Data: Save categories/tags as JSON
  • Export Bookmarks: Save as HTML
  • Import Data: Restore from JSON
  • Clear Data: Remove all categories/tags

‚ùì FAQ

Q: Is it expensive to use?

A: Very affordable. Using gpt-4o-mini, most users spend less than $1/month.

Q: Is my data safe?

A: Yes! All data is stored locally. API key is encrypted. No data sent to external servers except when using AI features.

Q: Which browsers are supported?

A: All Chromium-based browsers: Chrome, Edge, Brave, Opera, etc.

Q: How to backup data?

A: In settings, click "Export Data" or "Export Bookmarks (HTML)". Backup regularly.
