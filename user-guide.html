📚 AI Bookmark Manager 使用说明

AI Bookmark Manager 是一款智能书签管理浏览器扩展，使用 AI 技术自动分类、整理和搜索您的书签，让书签管理变得简单高效。

🚀 安装与配置

1. 安装扩展

  1. 打开 Chrome/Edge 浏览器
  2. 访问 chrome://extensions/ 或 edge://extensions/
  3. 开启右上角的"开发者模式"
  4. 点击"加载已解压的扩展程序"
  5. 选择扩展文件夹
  6. 扩展图标出现在工具栏，安装完成！

2. 配置 OpenAI API

获取 API Key

  1. 访问 OpenAI Platform
  2. 注册或登录账号
  3. 创建新的 API Key
  4. 复制 API Key（格式：sk-...）
  1. 点击扩展图标，自动打开配置页面
  2. 粘贴 API Key
  3. 选择模型：推荐 gpt-4o-mini（性价比最高）
  4. 点击"测试连接"确保配置正确
  5. 点击"保存设置"
✅ 配置完成！ 现在可以开始使用 AI 功能了。

📖 基础使用

保存书签

有三种方式保存当前页面为书签：

  • 快捷键：按 Cmd+Shift+S (Mac) 或 Ctrl+Shift+S (Windows)
  • 侧边栏：打开侧边栏，点击"保存当前页面"按钮
  • 浏览器原生：使用浏览器自带的书签功能

查看书签

  1. 点击扩展图标打开侧边栏
  2. 书签按文件夹/分类分组显示
  3. 点击文件夹名称展开/折叠
  4. 点击书签打开网页
💡 提示： 书签默认按照您的浏览器文件夹结构分类。使用 AI 分类后，会优先显示用户文件夹，其次是 AI 分类。

搜索书签

  1. 在侧边栏顶部的搜索框输入关键词
  2. 实时显示匹配结果
  3. 支持搜索标题、URL、标签和分类
  4. 勾选"AI 搜索"启用语义搜索

筛选书签

  • 按分类筛选：在分类下拉框选择类别
  • 按标签筛选：在标签下拉框选择标签
  • 组合筛选：同时使用分类和标签筛选
  • 清除筛选：点击"清除筛选"按钮恢复

🤖 AI 智能功能

AI 自动分类

功能说明

AI 会分析每个书签的标题和 URL，自动生成：

  • 分类：如技术、新闻、娱乐、购物等
  • 标签：3-5 个相关标签
  • 描述：一句话摘要

使用方法：

  1. 打开侧边栏
  2. 点击"AI 分类"按钮
  3. 确认对话框
  4. 等待 AI 处理（显示进度条）
  5. 完成后书签自动按分类分组
💡 成本说明： 使用 gpt-4o-mini 模型，分类 100 个书签约花费 $0.01-0.02，非常便宜。

AI 语义搜索

与普通关键词搜索不同，AI 语义搜索能理解您的意图：

  • 搜索"学习前端" → 找到所有前端相关资源
  • 搜索"买东西" → 找到所有购物网站
  • 搜索"看新闻" → 找到所有新闻网站

使用方法：

  1. 在搜索框输入查询
  2. 勾选"AI 搜索"
  3. 等待 2-5 秒
  4. 查看匹配结果

⚙️ 高级功能

书签管理页面

点击侧边栏的"管理页面"按钮，打开完整的书签管理界面。

功能包括：

  • 批量操作：勾选多个书签，批量分类或删除
  • 添加书签：手动添加新书签
  • 编辑书签：修改标题、URL、分类、标签
  • 查找重复：检测重复的书签
  • 查找失效：检测失效的书签（如 127.0.0.1）
  • 视图切换：列表视图或网格视图
  • 排序：按标题、日期、URL 排序

查找失效书签

自动检测以下类型的失效书签：

  • 本地链接（localhost, 127.0.0.1）
  • 局域网链接（192.168.x.x, 10.x.x.x）
  • 文件链接（file://）
  • Chrome 内部链接（chrome://）
  • 临时/测试链接（包含 temp, test, example.com）

使用方法：

  1. 打开管理页面
  2. 点击"查找失效书签"
  3. 查看检测结果
  4. 确认后批量删除
⚠️ 注意： 删除操作不可恢复，请谨慎操作。建议先导出备份。

数据管理

在配置页面的"数据管理"区域：

  • 导出数据：导出分类和标签为 JSON 文件
  • 导出书签：导出为标准 HTML 格式
  • 导入数据：从 JSON 文件恢复数据
  • 清除数据：清除所有分类和标签（不删除书签）

主题设置

支持三种主题模式：

  • 浅色模式：明亮清爽
  • 深色模式：护眼舒适
  • 跟随系统：自动切换（默认）

❓ 常见问题

Q: 没有 OpenAI API Key 怎么办？

A: 访问 OpenAI Platform 注册并创建 API Key。新用户通常有免费额度。不使用 AI 功能的话，也可以正常使用基础的书签管理功能。

Q: API 调用会很贵吗？

A: 使用 gpt-4o-mini 模型非常便宜，分类 100 个书签约 $0.01-0.02。一般用户每月花费不到 $1。

Q: 数据安全吗？

A: 完全安全！所有数据存储在本地浏览器，API Key 加密保存，不会上传到任何服务器。只有在您主动触发 AI 功能时才会调用 OpenAI API。

Q: 书签会被删除吗？

A: 不会。扩展只会添加分类和标签等元数据，不会删除或修改您的原始书签（除非您主动删除）。

Q: 支持哪些浏览器？

A: 支持所有基于 Chromium 的浏览器：Chrome、Edge、Brave、Opera 等。暂不支持 Firefox。

Q: AI 分类不准确怎么办？

A: 可以手动编辑书签的分类和标签。在管理页面点击编辑按钮，修改为正确的分类。

Q: 如何备份数据？

A: 在配置页面点击"导出数据"保存 JSON 文件，或点击"导出书签 (HTML)"导出标准格式。建议定期备份。

Q: 扩展占用资源多吗？

A: 非常轻量。内存占用 < 50MB，CPU 占用几乎为 0（空闲时）。不会影响浏览器性能。

Q: 如何卸载？

A: 在 chrome://extensions/ 找到扩展，点击"移除"。注意：卸载会删除所有分类和标签数据，但不会删除浏览器原生书签。

Q: 遇到问题怎么办？

A: 可以通过以下方式获取帮助：

  • 查看浏览器控制台的错误信息
  • 检查 API Key 配置和网络连接
  • 访问 GitHub 项目
  • 发送邮件到 simonguo.2009@gmail.com
