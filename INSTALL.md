# 安装指南

## 前置要求

- Chrome、Edge 或其他基于 Chromium 的浏览器
- OpenAI API Key（从 [OpenAI Platform](https://platform.openai.com/api-keys) 获取）

## 安装步骤

### 1. 生成图标文件

首先需要生成扩展所需的图标文件：

1. 在浏览器中打开 `generate-icons.html`
2. 点击"下载全部"按钮
3. 将下载的 4 个 PNG 文件（icon16.png, icon32.png, icon48.png, icon128.png）移动到 `icons/` 文件夹

**或者**，如果你安装了 ImageMagick：

```bash
chmod +x generate-icons.sh
./generate-icons.sh
```

### 2. 加载扩展到浏览器

#### Chrome / Edge

1. 打开浏览器，访问扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 启用右上角的"开发者模式"

3. 点击"加载已解压的扩展程序"

4. 选择 `bookmark-ai-extension` 文件夹

5. 扩展图标应该出现在浏览器工具栏中

### 3. 配置 OpenAI API

1. 点击扩展图标，或右键点击图标选择"选项"

2. 在设置页面中：
   - 输入你的 OpenAI API Key
   - 选择模型（推荐使用 `gpt-4o-mini`，性价比最高）
   - 点击"测试连接"确保配置正确
   - 点击"保存设置"

3. 配置完成！

## 使用方法

### 快捷键

- `Cmd+Shift+S` (Mac) / `Ctrl+Shift+S` (Windows/Linux) - 快速保存当前页面
- `Cmd+Shift+B` (Mac) / `Ctrl+Shift+B` (Windows/Linux) - 打开书签侧边栏

### 侧边栏

1. 点击扩展图标打开侧边栏
2. 浏览树形结构的书签
3. 使用搜索框查找书签
4. 点击"AI 分类"按钮对所有书签进行智能分类

### 管理页面

1. 在侧边栏点击"管理页面"按钮
2. 或右键点击扩展图标 → 选项
3. 在管理页面可以：
   - 批量编辑书签
   - 查找重复书签
   - 删除不需要的书签
   - 调整书签位置

## 故障排除

### 扩展无法加载

- 确保所有文件都在 `bookmark-ai-extension` 文件夹中
- 确保图标文件已生成并放在 `icons/` 文件夹
- 检查浏览器控制台是否有错误信息

### AI 功能不工作

- 确保已正确配置 OpenAI API Key
- 在设置页面点击"测试连接"验证
- 检查 API Key 是否有足够的额度
- 确保网络连接正常

### 书签未显示

- 点击刷新按钮重新加载书签
- 检查浏览器是否有书签权限
- 尝试重新加载扩展

## 隐私与安全

- ✅ 所有书签数据存储在本地浏览器中
- ✅ OpenAI API Key 加密存储在本地
- ✅ 仅在用户主动触发时调用 OpenAI API
- ✅ 不会收集或上传任何用户数据
- ✅ 开源代码，可自行审查

## 卸载

1. 打开扩展管理页面
2. 找到 "AI Bookmark Manager"
3. 点击"移除"
4. 确认卸载

**注意**：卸载扩展会删除所有分类和标签数据，但不会删除浏览器原生书签。
